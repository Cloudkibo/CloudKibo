### User Entity

**/api/users/**
Get list of users

HTTP Method: GET

**/api/users/:id**
Deletes the user

HTTP Method: DELETE

**/api/usesr/me**
Get user Profile

HTTP Method: GET

**/api/users/:id/password**
Changes user's password

HTTP Method: PUT

**/api/users/:id**
Get single user

HTTP Method: GET

**/api/users/**
Create new user

HTTP Method: POST

**/api/users/userimage/update**
Update user image

HTTP Method: POST

**/api/users/userimage/:image**
Get user image

HTTP Method: GET

**/api/users/update**
Change user profile

HTTP Method: PUT

**/api/users/searchbyusername**

HTTP Method: POST

**/api/users/searchbyuseremail**
